## Ban
#### Permission level:
Admin+
#### Aliases:
`Ban`
#### Description:
Bans a user from the guild. Will ask for confirmation.<br/>
Any extra text will be converted to the ban reason.
#### Usage Example:
`!ban @Guildbot#2193 Causes drama`
<br/>
<br/>
<br/>
## Kick
#### Permission level:
Admin+
#### Aliases:
`Kick`
#### Description:

Any extra text will be converted to the kick reason.
#### Usage Example:
`!kick @Guildbot#2193 Causes drama`
<br/>
<br/>
<br/>
## Purge Self
#### Permission level:
Moderator+
#### Aliases:
`Purge, PurgeSelf, Prune, PruneSelf`
#### Description:
Instantly delete up to 100 messages from a channel sent by the bot.
#### Usage Example:
`!purge 76` or `!prune 42`
<br/>
<br/>
<br/>
## Mass Delete
#### Permission level:
Moderator+
#### Aliases:
`MassDelete, MD`
#### Description:
Instantly delete up to 100 messages from a channel (including the command you just sent).<br/>
*I am not responsible for any information you may lose.*
#### Usage Example:
Usage of the command would simply be: `!massdelete 76` or `!md 42`
<br/>
<br/>
<br/>
